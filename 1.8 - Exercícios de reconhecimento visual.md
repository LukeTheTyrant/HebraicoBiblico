## 1. Exercício — Direção da leitura

Observe a sequência:

```text
א  ב  ג  ד  ה
```

Qual é a direção correta da leitura?

A resposta correta é:

```text
direita ← esquerda
```

Portanto, o movimento visual é:

```text
ה ← ד ← ג ← ב ← א
```

---

## 2. Exercício — Nome das letras

Tente identificar as letras abaixo:

```text
א  ב  ג  ד  ה
```

Resposta:

```text
א — Alef
ב — Bet
ג — Gimel
ד — Dalet
ה — He
```

```text
ט — Tet
י — Yod
```

## 6. Exercício — Último grupo

```text
ש  ת
```

Resposta:

```text

ת — Tav
```

---

## 7. Exercício — Letras finais

Relacione as formas normais com as finais:

```text
כ → ?
מ → ?
נ → ?
פ → ?
צ → ?
```

Resposta:

```text
כ → ך
מ → ם
נ → ן
פ → ף
צ → ץ
```

---

## 8. Exercício — Letras parecidas

Compare os pares:

```text
ד / ר
ה / ח
ב / כ
ו / ז
מ / ס
נ / ג
```

Diga o nome de cada letra.

Resposta:

```text
ד — Dalet / ר — Resh
ה — He / ח — Het
ב — Bet / כ — Kaf
ו — Vav / ז — Zayin
מ — Mem / ס — Samekh
נ — Nun / ג — Gimel
```
