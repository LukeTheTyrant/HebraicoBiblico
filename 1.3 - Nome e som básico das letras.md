## Objetivo da etapa

Associar cada letra hebraica a um **nome** e a um **som básico aproximado**.

Como seu objetivo é ler hebraico bíblico, você não precisa dominar todas as discussões de pronúncia.  
Nesta etapa, basta criar uma associação inicial:

```text
letra → nome → som aproximado
```

---

## 1. Observação importante sobre pronúncia

A pronúncia do hebraico bíblico pode variar conforme a tradição:

- pronúncia acadêmica;
- pronúncia judaica moderna;
- pronúncia sefardita;
- pronúncia asquenazita;
- reconstruções históricas.

Para seu objetivo inicial, usaremos uma pronúncia **didática e aproximada**, suficiente para leitura básica.

| ד | Dalet | d |
| ה | He | h leve |

| ח | Het | h forte, gutural |

| י | Yod | y/i |
| כ | Kaf | k ou kh |
| ל | Lamed | l |
| מ | Mem | m |
| נ | Nun | n |
| ס | Samekh | s |
| ע | Ayin | gutural; muitas vezes quase silenciosa para iniciantes |
| פ | Pe | p ou f |

| ק | Qof | q/k |
| ר | Resh | r |
| ש | Shin / Sin | sh ou s |
| ת | Tav | t |

## 3. Letras que podem ter dois sons

Algumas letras mudam de som dependendo de sinais que você estudará depois.

Por enquanto, apenas observe:

| Letra | Possíveis sons |
|---|---|
| ב | b / v |
| כ | k / kh |
| פ | p / f |
| ש | sh / s |
| ו | v / o / u |

Não tente resolver tudo agora.  
A etapa [1.7 - Letras begadkefat](<1.7 - Letras begadkefat.md>) explicará parte disso.

---

## 4. Letras quase silenciosas para iniciantes

Duas letras podem parecer “sem som” para quem está começando:

| Letra | Nome | Observação |
|---|---|---|
| א | Alef | frequentemente funciona como suporte para vogal |
| ע | Ayin | gutural difícil; pode ser tratada inicialmente como suporte vocálico |

Exemplo:

```text
אֱלֹהִים
```

A primeira letra é:

```text
א
```

Ela é Alef.  
Na leitura inicial, você perceberá mais a vogal associada a ela do que um som consonantal forte.

## 5. Sons úteis para começar a ler

Memorize primeiro estes sons mais simples:

| Letra | Nome | Som |
|---|---|---|
| ב | Bet | b |
| ג | Gimel | g |
| ד | Dalet | d |
| ז | Zayin | z |
| ט | Tet | t |
| ל | Lamed | l |
| מ | Mem | m |
| נ | Nun | n |
| ס | Samekh | s |
| ר | Resh | r |
| ת | Tav | t |

Essas letras são mais fáceis de associar a sons conhecidos em português.

## 6. Exemplos com palavras bíblicas

### אֱלֹהִים

```text
א  ל  ה  י  ם
```

| Letra | Nome | Som aproximado |
|---|---|---|
| א | Alef | apoio vocálico |
| ל | Lamed | l |
| ה | He | h |
| י | Yod | y/i |
| ם | Mem final | m |

Leitura aproximada:

```text
Elohim
```

---

### יוֹם

```text
י  ו  ם
```

| Letra | Nome | Som aproximado |
|---|---|---|
| י | Yod | y/i |
| ו | Vav | o/u ou v |
| ם | Mem final | m |

Leitura aproximada:

```text
Yom
```

Significado:

```text
dia
```

---

### טוֹב

```text
ט  ו  ב
```

| Letra | Nome | Som aproximado |
|---|---|---|
| ט | Tet | t |
| ו | Vav | o/u |
| ב | Bet | b/v |

Leitura aproximada:

```text
Tov
```

Significado:

```text
bom
```

---

## 7. Como praticar

Use este ciclo:

```text
ver letra → dizer nome → dizer som aproximado
```

Exemplo:

```text
ל
```

Resposta:

```text
Lamed — l
```

Outro exemplo:

```text
מ
```

Resposta:

```text
Mem — m
```

## 11. Mini-resumo

Nesta etapa, você associou as letras hebraicas a nomes e sons básicos.

Não é necessário pronunciar tudo perfeitamente agora.

A habilidade principal é:

```text
letra → nome → som aproximado
```

Exemplo:

```text
ל → Lamed → l
מ → Mem → m
ת → Tav → t
```
