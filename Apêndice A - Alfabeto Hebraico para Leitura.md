Servir como tabela rápida de consulta do alfabeto hebraico.

## 1. As 22 letras hebraicas

| Letra | Nome | Som aproximado |
|---|---|---|
| א | Alef | apoio vocálico / quase silenciosa |
| ב | Bet | b / v |
| ג | Gimel | g |
| ד | Dalet | d |
| ה | He | h leve |
| ו | Vav | v / o / u |
| ז | Zayin | z |
| ח | Het | h gutural |
| ט | Tet | t |
| י | Yod | y / i |
| כ | Kaf | k / kh |
| ל | Lamed | l |
| מ | Mem | m |
| נ | Nun | n |
| ס | Samekh | s |
| ע | Ayin | gutural / apoio vocálico |
| פ | Pe | p / f |
| צ | Tsade | ts |
| ק | Qof | q / k |
| ר | Resh | r |
| ש | Shin / Sin | sh / s |
| ת | Tav | t |

## 6. Letras begadkefat

```text
ב  ג  ד  כ  פ  ת
```

Essas letras podem ter comportamento especial de pronúncia, especialmente com dagesh.

---

## 7. Letras que costumam confundir

| Par | Atenção |
|---|---|
| ד / ר | Dalet e Resh |
| ה / ח | He e Het |
| ב / כ | Bet e Kaf |
| ו / ז | Vav e Zayin |
| מ / ס | Mem e Samekh |
| נ / ג | Nun e Gimel |

---

## 8. Exercício rápido

Tente identificar:

```text
א  מ  ל  ת  ש  ד  ר  ה  ח
```

Resposta:

```text
א — Alef
מ — Mem
ל — Lamed
ת — Tav
ש — Shin/Sin
ד — Dalet
ר — Resh
ה — He
ח — Het
```
