## Objetivo do apêndice

Oferecer um checklist semanal simples para manter constância no estudo do hebraico bíblico.

O objetivo é evitar travamento e manter progresso leve.

---

# Checklist semanal

## Semana de estudo

```text
Semana: __ / __ / ____
```

---

## 1. Leitura do alfabeto

- [ ] Revisei as 22 letras hebraicas.
- [ ] Revisei as letras finais.
- [ ] Revisei letras parecidas.
- [ ] Revisei guturais.
- [ ] Revisei begadkefat.

---

## 2. Vogais

- [ ] Revisei patah.
- [ ] Revisei qamats.
- [ ] Revisei segol.
- [ ] Revisei tsere.
- [ ] Revisei hireq.
- [ ] Revisei holam.
- [ ] Revisei qibbuts.
- [ ] Revisei shureq.
- [ ] Revisei sheva.

---

## 3. Vocabulário

- [ ] Revisei 10 palavras já conhecidas.
- [ ] Revisei 20 palavras já conhecidas.
- [ ] Aprendi 5 palavras novas.
- [ ] Agrupei palavras por tema.
- [ ] Revisei nomes divinos.
- [ ] Revisei partículas frequentes.
- [ ] Revisei verbos frequentes.

---

## 4. Leitura em voz alta

- [ ] Li palavras isoladas em voz alta.
- [ ] Li um versículo curto em voz alta.
- [ ] Repeti o mesmo versículo mais de uma vez.
- [ ] Li sem depender totalmente da transliteração.

---

## 5. Leitura de versículo

- [ ] Escolhi um versículo curto.
- [ ] Copiei ou consultei o texto hebraico.
- [ ] Identifiquei palavras conhecidas.
- [ ] Identifiquei partículas.
- [ ] Identifiquei substantivos.
- [ ] Identifiquei o verbo principal.
- [ ] Fiz uma tradução literal.
- [ ] Comparei com uma tradução católica.
- [ ] Escrevi uma observação teológica simples.

---

## 7. Regra contra travamento

Se eu travar, devo fazer apenas uma destas opções:

- [ ] Revisar 5 letras.
- [ ] Revisar 5 vogais.
- [ ] Revisar 5 palavras.
- [ ] Ler 1 versículo já conhecido.
- [ ] Fazer uma nota curta com apenas uma observação.

O objetivo é continuar, não fazer perfeito.

---

## 8. Fechamento da semana

- [ ] Mantive contato com o hebraico bíblico nesta semana.
- [ ] Fiz pelo menos uma revisão curta.
- [ ] Não tentei aprender tudo de uma vez.
- [ ] Avancei um pouco.
